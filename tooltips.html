Tooltips

Tooltips are used to provide information to the user about the purpose of an element or to indicate that an action is about to be cmpleted or has been completed.

For example, a tooltip can be placed over a button to indicate that the user can clear form contents

The tooltip should:

  • Appear next to the element so that it does not cover important content, use styles
  • Be used only as needed
  • Contain brief instructions, up to five words
