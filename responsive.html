Responsive Design

Guidelines

Layout refers to the arrangement of visual elements on a page so that the user can have a good experience regardless of viewport size. Page development should follow testing and reiteration of wireframes.

All pages must include:

  1. Headers containing an H1 page title with company name
  2. Navigation menu
    • Main navigation above the fold
    • Supplemental navigation below the fold in the footer
  3. Body with most important content above the fold
  4. Footer with CC logo
  5. On wider screens content cards will be displayed horizontally but vertically on smaller screens

Recommended Layouts

Code for cards and navigation are described in detail elsewhere on this site.
