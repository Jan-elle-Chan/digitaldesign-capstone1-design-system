Whitespace

Whitespace, also called negative space consists of spaces, tabs and linebreaks between text, as well as the space between the elements on a page.

Whitespace in development files and webpages must both be considered.

Whitespace in HTML files

  • Extra spaces between a string will be ignored, use special character sequences or tags to introduce whitespace
  • Use whitespace to indent code for readability
How to Add Whitespace to HTML document
Tag or Character Use
<br> To move text to next line, like a carriage return
<p> To create a paragraph break within a block of text
&nbsp; To move text to next line, like a carriage return
<pre> To display text exactly as it appears bewteen the tags
